# Sacrifunk Lab Tools

Interactive music theory tools built with React, hosted at [sacrifunk.com/lab](https://sacrifunk.com/lab).

## Tools

| File | Tool |
|------|------|
| `sacrifunk_app.jsx` | Main Sacrifunk app |
| `sacrifunk_scale_explorer.jsx` | Scale Explorer — explore the 5 original Sacrifunk scales |
| `sacrifunk_freq_explorer.jsx` | Frequency Explorer — browse all notes across 4 tuning systems |
| `sacrifunk_freq_designer.jsx` | Frequency Designer — design custom frequency combinations |
| `sacrifunk_vortex_explorer.jsx` | Vortex Explorer — vortex mathematics visualization |
| `sacrifunk_vortex_v2.jsx` | Vortex Explorer v2 |
| `sacrifunk_cymatics_lab.jsx` | Cymatics Lab — visual frequency patterns |
| `five_laws_tutorial.jsx` | Five Laws Tutorial — interactive guide to the Five Laws |
